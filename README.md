# Sidebar Assignment

A modern, responsive sidebar navigation assignment with social SVG icons and clear front-page background.

## Features

- **Responsive Navbar**  
  - Desktop: Full navbar with logo, menu links (Home, About, Projects, Contact), and social SVG icons.  
  - Mobile: Logo with toggle button; sidebar slides in when toggle is clicked.

- **Sidebar**  
  - Glass-style transparent background with blur effect.  
  - Contains menu links and social SVG icons.  
  - Close button to hide sidebar on mobile.

- **Social Icons**  
  - Real SVGs for Facebook, GitHub, LinkedIn, YouTube.  
  - Hover effect: icons move slightly upward and show real brand colors.

- **Background Image**  
  - Clear, front-page background image (not blurred).  

- **Fully Responsive**  
  - Works perfectly on Desktop, Tablet, and Mobile.

## How to Use

1. Clone or download the repository.  
2. Open `index.html` in a browser.  
3. On mobile view, click the toggle button to open/close the sidebar.  
4. Hover over social icons to see hover animation and color effect.
